# Accessible Chapter 1 (preview)

This is an accessible, screen-reader-friendly version of **Chapter 1: The Interactive
Computing Environment**, prepared as a pilot. The teaching content matches the main
course site; the differences are in *how you do the work*:

- a workflow built on **VS Code and Python scripts** instead of the in-browser
  JupyterLab/Colab notebook interface, which is hard to use with a screen reader;
- a new [Setting up an accessible workflow](lectures_DS/computing_env/accessible_setup.md)
  page and a [Working with Python scripts](lectures_DS/computing_env/working_with_scripts.md)
  page that replace the notebook-focused material;
- figures replaced with text descriptions, and the hands-on activities rewritten for this
  setup.

**Start with the [Overview](lectures_DS/computing_env/overview.md)**, which lists the
pages in order.
